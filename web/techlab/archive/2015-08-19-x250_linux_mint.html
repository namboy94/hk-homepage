Setting up a Lenovo Thinkpad X250 with Linux Mint 17.1

Setting up a Lenovo Thinkpad X250 with Linux Mint 17.1 is fairly easy, most things work out of the box without a hitch. Some things however, for example the Trackpoint buttons, need some tweaking to work as intended.

First off, update the Linux Kernel to version 3.19 or higher. This enables the Trackpoint buttons and vertical two-finger scrolling. The vertical two-finger scrolling may have to be enabled in Linux Mint's settings.

Afterwards, install the Intel graphics drivers using the following terminal commands:

sudo add-apt-repository ppa:paulo-miguel-dias/mesa
sudo add-apt-repository ppa:xorg-edgers/ppa
sudo apt-get update
sudo apt-get dist-upgrade
Afterwards, reboot the PC and the drivers should be up and running.
To use the Smartcard Reader, first install the required software with:

sudo apt-get install libusb-dev libusb++-0.1-4c2 libccid pcscd libpcsclite1 libpcsclite-dev libpcsc-perl pcsc-tools
To start the card reader, run:

pcsc_scan
