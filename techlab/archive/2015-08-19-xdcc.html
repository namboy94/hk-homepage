Downloading Files via XDCC

XDCC is a file transfer protocol used in conjunction with IRC (Internet Relay Chat). Ideally, it offers fast transfer times, pausable downloads and high quality media.

To download via IRC/XDCC, you'll need an XDCC-able IRC client. There are many of these available, I recommend Hexchat for a GUI solution and irssi for a CLI alternative.

To start a download, you'll typically have to connect to a specific IRC server and join a channel. To do this, enter the following in the text field of the IRC client:

/server irc.server.domain
/join #channel
Once you've successfully connected to the server and channel, you're ready to download. To start a download, you'll have to have access to a bot's packlist. There are many sites online that keep track of multiple bot's packlist and automatically format the download string for you. The download string should be formated like this:

/msg bot xdcc send #packnumber
The download should start now, depending on your configuration you'll be prompted to accept the file transfer.

That's basically it!
