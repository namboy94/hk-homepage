Installing proprietary Nvidia drivers on Linux Mint 17.2

If you have a graphics card and run Linux, you'll want to use the proprietary drivers instead of the community drivers for increased performance. While the Nouveau driver is fine for everyday tasks, anything more demanding essentially requires the proprietary Nvidia driver. I've also had issues using a monitor via DisplayPort without the proprietary driver.

This tutorial assumes a freshly installed Linux Mint 17.2 system. The methods described here may (or in fact should) work in other situations, but it's not guaranteed.

First off, download the appropriate driver for your specific GPU model from Nvidia's website and save the file to ~/Downloads .

Before we do anything, we'll want to get our system up to date. To do this, enter the following commands in the terminal:

sudo apt-get update
sudo apt-get dist-upgrade
sudo reboot
Afterwards, we'll blacklist the Nouveau driver:

echo "blacklist nouveau" | sudo tee -a /etc/modprobe.d/blacklist.conf
sudo reboot
After the reboot, do not log in! Enter the key combination

CTRL-ALT-F1
Enter the following commands after logging in using the tty1 terminal:

sudo service mdm stop
cd Downloads
sudo apt-get purge nvidia*
chmod +x Driver.run
sudo sh Driver.run
sudo apt-get install nvidia-prime
sudo reboot
The driver should now have successfully installed, to verify this, enter

inxi -G
in a terminal.

Enjoy your Nvidia driver!
